# backendTiendaEnLinea
# Mi API Backend

API REST hecha con Express, TypeScript y SQL Server.

## Instalación

\`\`\`bash
npm install
\`\`\`

## Configuración

Crea un archivo .env con estas variables:

DB_USER=
DB_PASSWORD=
DB_SERVER=
DB_DATABASE=
DB_PORT=
PORT=

## Ejecutar en desarrollo

\`\`\`bash
npm run dev
\`\`\`
